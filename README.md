Starting from some pictures of a cloth taken at different angles it will generate a textured mesh.

It uses [rembg](https://github.com/danielgatis/rembg), [MV-Adapter](https://github.com/huanngzh/MV-Adapter) and [Hunyuan3D-2](https://github.com/Tencent-Hunyuan/Hunyuan3D-2) respectively for each phase of the processing.
